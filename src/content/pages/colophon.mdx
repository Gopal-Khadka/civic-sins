A short note on how this site is built and who to thank.

## Art

Character art is generated with [Comicgen](https://gramener.com/comicgen/).
Language lives in the page, not the drawing, so a strip can be retold in any
tongue by swapping the text alone.

## Type

Set in Public Sans for headlines and body, and Space Grotesk for the stamped
labels. Both are open-source variable fonts, self-hosted so nothing loads from a
third party.

## Build

Built with TanStack Start, React, and TypeScript. Styled with Tailwind CSS and
Base UI primitives, with Phosphor for the few icons. Content is written in MDX.
Hosted on Netlify.

## Made by

Written, drawn, and built by Gopal Khadka. The source is open on
